Regex in JavaScript

Regular expressions are used in programming languages to match strings. You create patterns to help you do that matching.

.test() method takes the regex and applies it to the string(which is placed inside the parantheses), and returns true or false if your pattern finds something or not.

You can extract matches with the .match() method. Apply the method on a string and pass in the regex inside the parantheses

You can search for multiple patters using the alternation or OR operator: |

To ignore case use the i flag /regexhere/i

To search or extract a pattern more than once, use the g flag.

The wildcard character . will match any one character. It is also called dot and period.

Character classes can be placed inside []. They allow you to define a group of characters you wish to match. Specify a range of characters using the hyphen - . Eg. /[a-z0-9]/ig;

Negated character sets. To create these, place a caret (^) after the opening bracket and before the characters you do not want to match.

To match a character that occurs once or more in a row, use the plus (+) character. This means that it occurs atleast once and may be repeated. The character or pattern has to be present consecutively.

To match a character that occurs zero or more times, use the asterisk (*) character.

greedy match v/s lazy match. Regex are by default greedy. Use the character ? to change it to lazy matching.
